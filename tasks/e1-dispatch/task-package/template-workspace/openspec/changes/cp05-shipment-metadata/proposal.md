## Why
Partners need to know which carrier handled each shipment so they can track deliveries and process claims.

## What Changes
- When a line is shipped, the carrier name and tracking number may be recorded against it.
- Exported orders include carrier and tracking on each shipped line when recorded; lines without this information export exactly as before.
- Re-importing an exported order preserves carrier and tracking on each line.
- An order's status is not affected by whether carrier information is present.
