# CP08: Cancel after shipment

Operations cancels orders even after some lines have shipped. An order cancelled after at least one line has shipped is now reported as `cancelled_partial` everywhere a status is shown; such orders previously appeared as `cancelled`. An exported `cancelled_partial` order carries `"requires_refund":true`; re-importing it preserves that flag.

## Scenarios

#### Scenario: Partially shipped order is cancelled
- **GIVEN** a paid order with one line shipped
- **WHEN** the order is cancelled
- **THEN** everywhere a status is shown it is cancelled_partial and the export carries requires_refund:true

#### Scenario: Clean cancellation is unchanged
- **GIVEN** a paid order with no lines shipped
- **WHEN** the order is cancelled
- **THEN** the status is cancelled with no requires_refund flag
